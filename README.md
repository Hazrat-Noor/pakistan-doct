## 🌟 Overview

**Kisan Dost Pakistan (کسان دوست پاکستان)** is a digital agriculture platform engineered specifically for Pakistan's agricultural community. It equips farmers, agronomy students, researchers, field officers, and gardeners with **science-backed, step-by-step crop cultivation guides**, **soil management protocols**, **pest & disease remedies**, **fertilizer calculators**, and **seasonal crop calendars**—completely accessible in both **English and Urdu**.

Whether cultivating staples like Wheat and Basmati Rice in Punjab, Cotton and Sugarcane in Sindh, or Orchards in Khyber Pakhtunkhwa and Balochistan, **Kisan Dost Pakistan** delivers actionable field intelligence right to your screen.

🌐 **Deployed Web Application:** [https://pakistan-doct.vercel.app/](https://pakistan-doct.vercel.app/)

## 🔥 Core Features

### 1. 🌾 Complete 23 Major Pakistani Crops Library
Detailed cultivation guides covering **23 major food crops, cash crops, pulses, oilseeds, and fruits**:
- **Climatic Requirements:** Ideal temperature, soil pH, and rainfall conditions.
- **Varieties:** Approved certified Pakistani seeds (e.g., Akbar-19, Super Basmati, FH-142).
- **Land Prep & Sowing:** Depth, spacing (R-R & P-P), and land leveling instructions.
- **Fertilizer Schedule:** Phase-wise application of DAP, Urea, SOP, and Micronutrients (Zinc, Boron).
- **Irrigation Timings:** Crucial growth stages (Crown Root Initiation, Flowering, Grain Filling) for watering.
- **Pest & Disease Remedies:** Detailed symptom descriptions with eco-friendly & chemical solutions.
- **Harvesting & Storage:** Maturity indicators and grain moisture management.

### 2. 🧮 Precision Farm Seed & Fertilizer Calculator
Input your field size in **Acres** and select your crop to get instant recommendations:
- **Seed Rate:** Exact kilograms of certified seed needed for your farm size.
- **DAP Bags (Phosphorus):** Basal dose requirements at sowing time.
- **Urea Bags (Nitrogen):** Split application schedule during vegetative stages.

### 3. 📅 Interactive Pakistani Crop Calendar
Month-by-month agricultural schedule spanning January through December:
- **Sowing Window:** Active crops to sow during current months.
- **Harvesting Window:** Crop maturity timelines for optimal market pricing.
- **Canal Water Priority:** Critical irrigation advisory per month.
- **Seasonal Division:** Filter by **Rabi (Winter)**, **Kharif (Summer)**, and **Zaid (Spring/Autumn)**.

### 4. 🎙️ Kisan Awaaz (Voice Assistant UI)
Simulated voice assistant supporting voice prompts and audio playback:
- Ask crop questions in simple spoken English or Urdu.
- Pre-loaded instant voice answers for quick farm queries (e.g., *"How much DAP for 5 acres of wheat?"*).
- Spoken audio simulation with visual equalizer animations.

### 5. 📚 Agricultural Learning Hub & FAQs
- **Guides:** Soil health, seed treatment, drip irrigation, pest management, and smart farming.
- **FAQs:** 10+ verified Q&As answering common field problems (e.g., controlling wheat rust, rice blast, or termite control).
- **Pakistan Agri Facts:** Highlighting Pakistan's top global rankings in Basmati rice exports, sugarcane production, and canal network architecture.

## 🚀 Quick Start & Local Setup

To run **Kisan Dost Pakistan** locally on your computer:

### Prerequisites
- **Node.js**: v18.0.0 or higher
- **npm**: v9.0.0 or higher

### Steps

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/your-username/kisan-dost-pakistan.git
   cd kisan-dost-pakistan
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:3000`.

4. **Build for Production:**
   ```bash
   npm run build
   ```
